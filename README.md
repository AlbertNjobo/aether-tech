# Aether Tech

Aether Tech is a Vite + React + TypeScript site showcasing a futuristic research studio. It includes a cinematic landing experience, animated sections, and an MDX-driven articles library.

## Highlights
- Immersive landing sections built with Tailwind and motion-heavy components.
- 3D/graphics support via Three.js, React Three Fiber, and P5.
- Research articles delivered as MDX with a dedicated index.
- Multi-page routing with `react-router-dom`.

## Tech stack
- Vite + React 19 + TypeScript
- Tailwind CSS + PostCSS
- MDX via `@mdx-js/rollup`
- Three.js, React Three Fiber, and Framer Motion

## Content
- Landing layout: `src/components/`
- Pages: `src/pages/`
- Articles: `src/content/articles/`

## Run locally
```bash
npm install
npm run dev
```
